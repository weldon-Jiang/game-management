```javascript
// auth.js - 登录请求
```

### 2.3 WebSocket 实时通信

#### Agent 连接流程
```
Agent 启动
    ↓
WSClient.connect(ws://host:port/{agentId}?agentSecret={secret})
    ↓
AgentWebSocketEndpoint.onOpen()
    ↓
validateAgent(agentId, agentSecret)
    ↓
AGENT_SESSIONS.put(agentId, session)
    ↓
sendMessage("connected", {...})
    ↓
broadcastToAdmins("agent_online", {...})
```

#### 心跳机制
- **Agent → 后端**：每 30 秒发送 heartbeat
- **后端 → Agent**：回复 heartbeat_ack
- **断开检测**：session.setMaxIdleTimeout(120000)

#### 更新状态
- ✅ 心跳机制完整
- ⚠️ 后端没有定时检查 Agent 心跳超时的逻辑（建议添加）

## 九、部署步骤

### 9.1 后端部署

```bash
# 1. 编译打包
cd bend-platform
mvn clean package -DskipTests

# 2. 配置环境变量 (必须使用 JDK 17)
export JAVA_HOME=/path/to/jdk-17
export PATH=$JAVA_HOME/bin:$PATH

# 3. 启动后端
java -jar target/bend-platform-1.0.0.jar
```

### 9.2 前端部署

```bash
# 1. 安装依赖
cd bend-platform-web
npm install

# 2. 开发环境运行
npm run dev

# 3. 生产环境构建
npm run build
```

### 9.3 Agent 部署

```bash
# 1. 安装 Python 依赖
cd bend-agent
pip install -r requirements.txt

# 2. 配置 Agent
cp config.example.yaml config.yaml
# 编辑 config.yaml 填写后端地址和认证信息

# 3. 启动 Agent
python -m bend_agent
```
