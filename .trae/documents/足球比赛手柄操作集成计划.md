## 一、分析总结

### 1.1 当前架构分析

**已有组件**：
1. **场景检测器** (`streaming_scene_detector.py`)
   - 支持多区域模板匹配
   - 返回 `SceneMatchResult(scene_id, matched, confidence, template_results)`
   - 场景配置定义在 `scene_schemas.py`

2. **手柄控制器** (`xbox_gamepad.py`)
   - 支持pygame手柄读取
   - 提供 `GamepadSignal` 信号类

3. **控制器协议** (`controller_protocol.py`)
   - `ControllerSignal` 类用于手柄信号传输
   - 支持按钮、扳机、摇杆设置

4. **Streaming项目分析结论**：
   - ✅ UI导航场景识别：已实现（xsrpst.py）
   - ✅ 手柄控制协议：已实现（xsrputil.py）
   - ❌ **比赛进行中的球员识别：未实现**
   - ❌ **传球/射门等智能决策：未实现**
   - ⚠️ **Payload中定义了frame_objects字段，但未实际使用**

**缺少组件**：
1. 足球比赛专用场景配置（进球、角球、任意球等）
2. **比赛画面球员位置识别模块**

## 二、Streaming项目分析

### 2.1 Streaming项目的能力范围

根据 `xsrpst.py` 和 `payload.py` 分析：

**已实现功能**：

3. **数据传输框架**
   - `PayloadFrame` 包含 `frame_objects` 字段
   - 支持帧数据上传到后端

**未实现功能**：
1. **球员位置识别** - 没有任何CV/AI代码
2. **足球位置检测** - 无目标检测代码
3. **比赛动作决策** - 无任何游戏AI逻辑

### 3.2 球员识别技术方案

**方案A：模板匹配（简单场景）**
- 使用预定义模板匹配球员颜色/形状
- 优点：实现简单，资源占用低
- 缺点：只适用于特定球队/阵型

**方案B：传统CV（颜色分割）**
- 根据球衣颜色分割球员
- 优点：无需训练，速度快
- 缺点：容易受光照影响

**方案C：深度学习（推荐）**
- 使用YOLO等目标检测模型
- 优点：准确率高，泛化能力强
- 缺点：需要GPU资源

**推荐方案**：方案B + 方案C 混合
- 简单场景用颜色分割
- 复杂场景用YOLO检测

### 3.3 比赛动作决策模块

```python
class MatchActionDecider:
    """
    比赛动作决策器
    
    功能：
    1. 分析当前比赛态势
    2. 决策下一步动作
    3. 生成手柄操作序列
    """
    
    def decide_action(
        self, 
        frame_data: MatchFrameData,
        game_state: GameState
    ) -> List[ActionStep]:
        """
        决策动作
        
        参数：
        - frame_data: 画面分析结果
        - game_state: 当前游戏状态
        
        返回：
        - 动作步骤列表
        """
        
        # 判断态势
        if frame_data.has_ball:
            # 我方持球 → 进攻决策
            return self._attack_decision(frame_data)
        else:
            # 对方持球 → 防守决策
            return self._defend_decision(frame_data)
    
    def _attack_decision(self, frame_data) -> List[ActionStep]:
        """进攻决策"""
        ball_pos = frame_data.ball_position
        
        # 判断距离球门的距离
        distance_to_goal = self._calc_distance_to_goal(ball_pos)
        
        if distance_to_goal < SHOOT_DISTANCE:
            # 可射门
            return [
                ActionStep(buttons=BUTTON_B, duration_ms=200),  # 射门
            ]
        elif distance_to_goal < PASS_DISTANCE:
            # 可传球
            return [
                ActionStep(buttons=BUTTON_A, duration_ms=150),  # 传球
            ]
        else:
            # 继续带球
            return [
                ActionStep(buttons=BUTTON_RB, duration_ms=300),  # 加速
                ActionStep(left_stick=(0, -1), duration_ms=500),  # 前进
            ]
    
    def _defend_decision(self, frame_data) -> List[ActionStep]:
        """防守决策"""
        # 切换到防守球员
        return [
            ActionStep(buttons=BUTTON_LB, duration_ms=100),  # 呼叫球员
            ActionStep(buttons=BUTTON_B, duration_ms=200),   # 抢断
        ]
```

#### 4.3.2 足球检测算法

```python
def detect_ball(frame):
    """
    检测足球位置
    
    方法：
    1. 颜色分割：白色/黑色圆形
    2. 圆形检测：霍夫圆变换
    3. 特征匹配：SIFT/ORB特征点
    """
    # 转HSV颜色空间
    hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)
    
    # 白色阈值（足球主色）
    lower_white = np.array([0, 0, 200])
    upper_white = np.array([180, 50, 255])
    mask = cv2.inRange(hsv, lower_white, upper_white)
    
    # 霍夫圆检测
    circles = cv2.HoughCircles(
        mask,
        cv2.HOUGH_GRADIENT,
        dp=1,
        minDist=20,
        param1=50,
        param2=30,
        minRadius=5,
        maxRadius=20
    )
    
    if circles is not None:
        # 返回最近的圆形（假设屏幕中心是足球）
        return (circles[0][0][0], circles[0][0][1])
    
    return None
```

#### 4.3.3 球员颜色分割

```python
def detect_players_by_color(frame):
    """
    根据球衣颜色检测球员
    
    假设：
    - 我方球衣：蓝色系
    - 对方球衣：红色系
    """
    hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)
    
    # 我方蓝色
    lower_blue = np.array([100, 100, 100])
    upper_blue = np.array([130, 255, 255])
    blue_mask = cv2.inRange(hsv, lower_blue, upper_blue)
    
    # 对方红色
    lower_red1 = np.array([0, 100, 100])
    upper_red1 = np.array([10, 255, 255])
    lower_red2 = np.array([170, 100, 100])
    upper_red2 = np.array([180, 255, 255])
    red_mask1 = cv2.inRange(hsv, lower_red1, upper_red1)
    red_mask2 = cv2.inRange(hsv, lower_red2, upper_red2)
    red_mask = cv2.bitwise_or(red_mask1, red_mask2)
    
    # 轮廓检测
    blue_contours, _ = cv2.findContours(blue_mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    red_contours, _ = cv2.findContours(red_mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    
    # 返回球员中心点
    blue_players = [get_centroid(c) for c in blue_contours if cv2.contourArea(c) > MIN_PLAYER_AREA]
    red_players = [get_centroid(c) for c in red_contours if cv2.contourArea(c) > MIN_PLAYER_AREA]
    
    return blue_players, red_players
```

#### 4.3.4 球门检测

```python
def detect_goals(frame):
    """
    检测球门位置
    
    方法：
    1. 白色大区域检测（球门）
    2. 门框线检测
    """
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    
    # 白色区域
    _, thresh = cv2.threshold(gray, 200, 255, cv2.THRESH_BINARY)
    
    # 找大轮廓（球门）
    contours, _ = cv2.findContours(thresh, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    
    goals = []
    for cnt in contours:
        area = cv2.contourArea(cnt)
        if area > GOAL_MIN_AREA:
            x, y, w, h = cv2.boundingRect(cnt)
            goals.append({
                'x': x, 'y': y,
                'width': w, 'height': h,
                'center': (x + w//2, y + h//2)
            })
    
    return goals
```

```python
    duration_ms: int = 100


class FootballController:
    """
    足球比赛控制器
    
    功能说明：
    - 加载动作配置
    - 执行单个动作
    - 执行动作序列
    - 与场景检测器联动
    
    使用方式：
        controller = FootballController()
        await controller.execute_action(FootballAction.SHOOT)
        await controller.execute_sequence("attack_sequence")
    """
    
    # Xbox按钮映射
    BUTTON_MAP = {
        'A': XboxButtonFlag.A,
        'B': XboxButtonFlag.B,
        'X': XboxButtonFlag.X,
        'Y': XboxButtonFlag.Y,
        'LB': XboxButtonFlag.L1,
        'RB': XboxButtonFlag.R1,
        'LS': XboxButtonFlag.L3,
        'RS': XboxButtonFlag.R3,
    }
    
    def __init__(self, config_path: str = None):
        self.logger = get_logger('football_controller')
        self.config = self._load_config(config_path)
        self.signal_sender = None  # 设置为实际的手柄信号发送器
    
    def _load_config(self, config_path: str) -> Dict:
        """加载动作配置"""
        if config_path is None:
            return self._get_default_config()
        # 从文件加载YAML配置
        ...
    
    def _get_default_config(self) -> Dict:
        """获取默认配置"""
        return {...}
    
    def set_signal_sender(self, sender):
        """设置手柄信号发送器"""
        self.signal_sender = sender
    
    async def execute_action(
        self, 
        action: FootballAction,
        duration_ms: int = None
    ) -> bool:
        """
        执行单个动作
        
        参数：
        - action: 动作枚举
        - duration_ms: 持续时间（覆盖配置）
        
        返回值：
        - True: 执行成功
        - False: 执行失败
        """
        action_config = self.config['basic_actions'].get(action.value)
        if not action_config:
            self.logger.error(f"未找到动作配置: {action.value}")
            return False
        
        duration = duration_ms or action_config['duration_ms']
        buttons = self._build_buttons(action_config['buttons'])
        
        signal = ControllerSignal()
        signal.buttons = buttons
        if 'triggers' in action_config:
            for trigger, value in action_config['triggers'].items():
                signal.set_trigger(trigger, value)
        
        # 发送手柄信号
        await self._send_signal(signal, duration)
        return True
    
    async def execute_sequence(
        self, 
        sequence_name: str,
        repeat: int = 1
    ) -> bool:
        """
        执行动作序列
        
        参数：
        - sequence_name: 序列名称
        - repeat: 重复次数
        
        返回值：
        - True: 执行成功
        - False: 执行失败
        """
        sequence = self.config['action_sequences'].get(sequence_name)
        if not sequence:
            self.logger.error(f"未找到动作序列: {sequence_name}")
            return False
        
        for _ in range(repeat):
            for step in sequence:
                action = step['action']
                duration = step['duration_ms']
                
                # 解析动作
                if action in self.BUTTON_MAP:
                    signal = ControllerSignal()
                    signal.buttons = self.BUTTON_MAP[action].value
                elif 'left_stick' in step:
                    signal = ControllerSignal()
                    x, y = step['left_stick']
                    signal.set_thumb('left', x, y)
                else:
                    # 从配置中查找动作
                    action_config = self.config['basic_actions'].get(action)
                    if action_config:
                        signal = ControllerSignal()
                        signal.buttons = self._build_buttons(action_config['buttons'])
                    else:
                        continue
                
                await self._send_signal(signal, duration)
        
        return True
    
    def _build_buttons(self, button_list: List[str]) -> int:
        """构建按钮位掩码"""
        buttons = 0
        for btn in button_list:
            if btn in self.BUTTON_MAP:
                buttons |= self.BUTTON_MAP[btn].value
        return buttons
    
    async def _send_signal(
        self, 
        signal: ControllerSignal, 
        duration_ms: int
    ):
        """发送手柄信号"""
        if self.signal_sender:
            await self.signal_sender(signal)
            await asyncio.sleep(duration_ms / 1000)
            # 释放所有按钮
            await self.signal_sender(ControllerSignal.zero())
    
    def angle_to_thumb(self, angle_degrees: float) -> Tuple[int, int]:
        """角度转换为摇杆坐标"""
        angle_rad = math.radians(angle_degrees)
        x = int(32767 * math.cos(angle_rad))
        y = int(32767 * math.sin(angle_rad))
        return x, y
```

```python
Scene Action Mapper - 场景动作映射器
=====================================

功能说明：

from typing import Dict, List, Optional, Tuple
from dataclasses import dataclass

from .streaming_scene_detector import StreamingSceneDetector, SceneMatchResult

@dataclass
class SceneAction:
    """场景动作定义"""
    scene_id: int
    action_type: str  # 'single', 'sequence', 'continuous'
    action_name: str
    priority: int = 0
    conditions: List[str] = None  # 执行条件


class SceneActionMapper:
    """
    场景动作映射器
    
    功能说明：
    - 管理场景与动作的映射关系
    - 根据场景执行相应动作
    - 支持动作序列和连续动作
    
    使用方式：
        mapper = SceneActionMapper(scene_detector, football_controller)
        await mapper.on_scene_detected(scene_result)
    """
    
    # 场景到动作的默认映射
    DEFAULT_MAPPINGS: Dict[int, SceneAction] = {
        # 100: SceneAction(100, 'sequence', 'kickoff_sequence'),
        # 101: SceneAction(101, 'continuous', 'attack_mode'),
        # 102: SceneAction(102, 'continuous', 'defend_mode'),
        # 103: SceneAction(103, 'sequence', 'celebrate'),
        # 104: SceneAction(104, 'sequence', 'match_end'),
    }
    
    def __init__(
        self,
        scene_detector: StreamingSceneDetector,
        football_controller: FootballController
    ):
        self.scene_detector = scene_detector
        self.football_controller = football_controller
        self.current_category = "menu"
        self.action_mappings = self.DEFAULT_MAPPINGS.copy()
    
    def register_action(self, scene_id: int, action: SceneAction):
        """注册场景动作"""
        self.action_mappings[scene_id] = action
    
    def get_action(self, scene_id: int) -> Optional[SceneAction]:
        """获取场景对应的动作"""
        return self.action_mappings.get(scene_id)
    
    async def on_scene_detected(
        self, 
        scene_result: SceneMatchResult
    ) -> bool:
        """
        场景检测回调
        
        参数：
        - scene_result: 场景检测结果
        
        返回值：
        - True: 动作执行成功
        - False: 执行失败或无对应动作
        """
        if not scene_result.matched:
            return False
        
        scene_id = scene_result.scene_id
        action = self.get_action(scene_id)
        
        if not action:
            return False
        
        # 更新场景分类
        self._update_category(scene_id)
        
        # 执行动作
        if action.action_type == 'single':
            enum_action = FootballAction(action.action_name)
            return await self.football_controller.execute_action(enum_action)
        
        elif action.action_type == 'sequence':
            return await self.football_controller.execute_sequence(action.action_name)
        
        elif action.action_type == 'continuous':
            # 持续动作模式
            return await self._execute_continuous_action(action)
        
        return False
    
    def _update_category(self, scene_id: int):
        """更新场景分类"""
        for category, scene_ids in SCENE_CATEGORY.items():
            if scene_id in scene_ids:
                self.current_category = category
                break

        return True
```
