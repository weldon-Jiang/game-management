# 游戏手柄操作自动化

## 目标
在 `_play_game` 函数中循环检测 Xbox 界面元素，根据不同界面模拟相应的手柄操作。

## 核心逻辑

```
循环检测 Xbox 界面
    ↓
检测到"暂停菜单" → 模拟按下 Start/Pause
    ↓
检测到"选项菜单" → 模拟按下 B 关闭菜单
    ↓
检测到"选择确认" → 模拟按下 A
    ↓
检测到"返回上级" → 模拟按下 B
    ↓
检测到"移动摇杆" → 模拟左摇杆移动
    ↓
... (根据实际界面扩展)
```

### 3. 界面检测循环

```python
def _play_game(self):
    """循环模拟游戏手柄操作"""
    logger.info("[playing] 开始游戏手柄操作循环...")

    while not self._failed:
        # 检测界面元素
        if self._wait_for_template("pause_menu", timeout=1, fail_on_timeout=False):
            logger.info("检测到暂停菜单，按下 Start")
            self._press_button(GamepadButton.START)
            continue

        if self._wait_for_template("confirm_dialog", timeout=1, fail_on_timeout=False):
            logger.info("检测到确认对话框，按下 A")
            self._press_button(GamepadButton.A)
            continue

        if self._wait_for_template("back_button", timeout=1, fail_on_timeout=False):
            logger.info("检测到返回按钮，按下 B")
            self._press_button(GamepadButton.B)
            continue

        # 无匹配，等待后继续检测
        time.sleep(0.5)
```

## 实施步骤

### 1. 添加 GamepadButton 枚举

### 2. 实现手柄操作方法

- `_release_all()` - 释放所有输入

### 3. 实现界面检测循环
