# Bend Platform 数据库连接配置

## DataGrip 连接配置

### 连接信息

| 配置项 | 值 |
|--------|-----|
| **主机** | `localhost` |
| **端口** | `3306` |
| **数据库** | `bend_platform` |
| **用户名** | `root` |
| **密码** | `123456` |
| **URL** | `jdbc:mysql://localhost:3306/bend_platform?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai` |

### DataGrip 连接步骤

1. 打开 DataGrip
2. 点击 `File` → `New` → `Data Source` → `MySQL`
3. 填写上述连接信息
4. 点击 `Test Connection` 测试连接
5. 连接成功后点击 `OK`

## 密码加密说明

### 加密方式

- **算法**: AES-256-ECB
- **密钥**: `bend-platform-secret-key`
- **存储格式**: `HEX(AES_ENCRYPT(明文, 密钥))`

### 加密后的密码示例

密码 `123456` 加密后存储为：`E236C28AC35D2486C21695F0049A0ED9`

### SQL 验证查询

```sql
-- 解密密码验证
SELECT
    AES_DECRYPT(UNHEX(password_hash), 'bend-platform-secret-key') AS decrypted_password
FROM merchant_user
WHERE username = 'admin';
```

## 常用 SQL

### 查询所有表

```sql
SHOW TABLES;
```

### 查询商户信息

```sql
SELECT * FROM merchant;
SELECT * FROM merchant_user;
```

### 查询串流账号状态

```sql
SELECT id, name, email, status, last_heartbeat FROM streaming_account;
```

### 查询 Xbox 主机状态

```sql
SELECT id, xbox_id, name, power_state, status, locked_by_agent_id FROM xbox_host;
```

### 查询错误日志

```sql
SELECT * FROM streaming_error_log ORDER BY created_at DESC LIMIT 20;
```
