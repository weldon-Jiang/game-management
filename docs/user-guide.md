## 登录注册

### 登录

1. 访问系统登录页面
2. 输入用户名和密码
3. 点击"登录"按钮
4. 登录成功后自动跳转首页

### 注册

1. 在登录页面点击"注册账号"
2. 填写注册信息：
   - 用户名
   - 密码
   - 确认密码
3. 点击"注册"完成

### 注意事项

- 用户名唯一，不可重复
- 密码至少6位字符
- 记住登录状态可保持7天免登录

## 商户管理

### 查看商户列表

1. 进入"商户管理"页面
2. 查看所有商户信息：
   - 商户名称
   - 联系电话
   - 状态（正常/过期/暂停）
   - 过期时间

### 创建商户

1. 点击"新建商户"按钮
2. 填写商户信息：
   - 商户名称
   - 联系电话
3. 点击"确定"保存

### 编辑商户状态

1. 在商户列表找到目标商户
2. 点击"状态"下拉框
3. 选择新状态：
   - `active` - 正常
   - `expired` - 过期
   - `suspended` - 暂停

## 流媒体账号管理

### 功能说明

管理 Xbox Game Pass 等流媒体服务账号，用于自动化登录和游戏控制。

### 查看账号列表

1. 进入"流媒体账号"页面
2. 查看账号信息：
   - 账号名称
   - 账号邮箱
   - 状态（正常/未激活/错误/离线）
   - 当前绑定Agent
   - 最后心跳时间

### 添加账号

1. 点击"添加账号"按钮
2. 填写账号信息：
   - 账号名称
   - 邮箱
   - 密码（加密存储）
   - 认证码（如有）
3. 点击"确定"保存

### 账号状态说明

| 状态 | 说明 | 处理方式 |
|------|------|----------|
| active | 正常 | 可正常使用 |
| inactive | 未激活 | 检查账号信息 |
| error | 错误 | 查看错误信息，重试 |
| offline | 离线 | Agent未连接 |

## 游戏账号管理

### 功能说明

管理 Xbox 游戏账号，与流媒体账号关联，用于游戏内操作。

### 查看游戏账号

1. 进入"游戏账号"页面
2. 查看账号信息：
   - 账号名称
   - Xbox Gamertag
   - 关联流媒体账号
   - 当前绑定Agent
   - 是否主账号
   - 使用优先级

### 绑定 Xbox 主机

1. 在游戏账号列表点击"绑定主机"
2. 选择可用的 Xbox 主机
3. 确认绑定

### 优先级设置

数值越高，优先级越高。系统会优先使用高优先级的账号进行任务。

## Agent 管理

### 查看 Agent 列表

1. 进入"Agent管理"页面
2. 查看 Agent 信息：
   - Agent ID
   - 商户
   - 主机地址
   - 版本
   - 状态（在线/离线/已卸载）

1. 点击 Agent 列表中的"查看任务"

   - 运行中的任务

### 状态说明

| 状态 | 说明 |
|------|------|
| online | Agent正常运行中 |
| offline | Agent已断开连接 |
| uninstalled | Agent已卸载 |

## 任务管理

### 查看任务列表

1. 进入"任务管理"页面
2. 使用筛选功能：
   - 任务状态
   - 任务类型
3. 查看任务信息：
   - 任务名称
   - 类型
   - 状态
   - 创建时间
   - 执行结果

### 创建任务

1. 点击"新建任务"按钮
2. 选择任务类型：
   - 模板匹配
   - 输入序列
   - 场景检测
   - 账号切换
   - 串流控制
   - 自定义
3. 配置任务参数
4. 选择目标Agent
5. 点击"确定"创建

### 任务状态

| 状态 | 说明 |
|------|------|
| pending | 待执行 |
| running | 执行中 |
| completed | 已完成 |
| failed | 已失败 |
| cancelled | 已取消 |

## 常见问题

### 登录问题

**Q: 忘记密码怎么办？**
A: 联系系统管理员重置密码

**Q: 登录显示无权限？**
A: 检查用户角色是否正确分配

### 任务问题

**Q: 任务一直处于pending状态？**
A: 检查是否有在线的Agent

**Q: 任务失败怎么办？**
A: 查看错误信息，根据提示调整任务参数

### Agent问题

**Q: Agent显示离线？**
A: 检查Agent程序是否运行，网络是否正常

**Q: 如何更换绑定的商户？**
A: 需要重新注册Agent，使用新的注册码
