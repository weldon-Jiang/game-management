## 1. 系统登录

### 1.1 访问系统

打开浏览器，访问平台地址：

```
http://服务器IP:3090
```

### 1.2 输入凭据

| 字段 | 说明 |
|------|------|
| 用户名 | 商户用户的登录名 |
| 密码 | 登录密码 |

### 1.3 角色说明

| 角色 | 说明 | 权限范围 |
|------|------|---------|
| **平台管理员** | 系统管理员 | 全部功能 |
| **商户管理员** | 商户所有者 | 本商户全部功能 |
| **操作员** | 普通员工 | 指定功能 |

### 1.4 登录后界面

登录成功后显示主界面，包含：
- 左侧菜单栏
- 顶部导航
- 主内容区

## 2. 平台管理员操作指南

### 2.1 商户管理

#### 查看商户列表

路径：**商户管理**

显示所有商户信息：
- 商户名称
- 联系电话
- 状态（正常/过期/暂停）
- 过期时间

#### 添加商户

1. 点击 **新建商户**
2. 填写信息：
   - 商户名称
   - 联系电话
3. 点击 **保存**

#### 编辑商户

1. 在商户列表点击 **编辑**
2. 修改信息
3. 点击 **保存**

#### 暂停/恢复商户

1. 在商户列表点击 **更多** → **暂停/恢复**
2. 确认操作

### 2.2 商户分组管理

路径：**商户分组**

用于商户的VIP等级/定价分组配置。

| 字段 | 说明 |
|------|------|
| VIP等级 | 等级名称 |
| 扣点折扣 | 价格折扣比例 |
| 解绑返还率 | 解绑时返还比例 |
| 每周解绑上限 | 最大解绑次数 |

### 2.3 用户管理

路径：**用户管理**

查看和管理所有商户用户。

#### 添加用户

1. 点击 **新建用户**
2. 填写信息：
   - 所属商户
   - 用户名
   - 手机号
   - 密码
   - 角色（商户管理员/操作员）
3. 点击 **保存**

#### 编辑用户

1. 在用户列表点击 **编辑**
2. 修改信息
3. 点击 **保存**

#### 重置密码

1. 点击用户 **重置密码**
2. 输入新密码
3. 点击 **确认**

### 2.4 激活码管理

路径：**激活码管理**

#### 生成激活码

1. 点击 **生成激活码**
2. 选择商户
3. 设置批次名称（可选）
4. 设置过期时间（可选）
5. 设置生成数量
6. 点击 **确定**

#### 导出激活码

1. 选择激活码批次
2. 点击 **导出**
3. 选择导出格式（Excel/TXT）

### 2.5 Agent版本管理

路径：**Agent版本**

#### 发布新版本

1. 点击 **发布新版本**
2. 填写信息：
   - 版本号
   - 下载URL
   - MD5校验码
   - 更新日志
   - 是否强制更新
3. 点击 **发布**

### 2.6 Agent实例管理

路径：**Agent实例**

查看所有Agent在线状态。

| 状态 | 说明 |
|------|------|
| 在线 | Agent正常运行 |
| 离线 | Agent未连接 |
| 已卸载 | Agent已卸载 |

## 5. Agent管理

### 5.1 Agent概述

Agent是运行在商户电脑上的自动化控制程序，负责：
- Xbox串流画面捕获
- 模板图像识别
- 自动化操作执行
- 账号切换管理

### 5.2 Agent安装

#### 收到安装包后

1. 解压 `BendAgent_v{x.x.x}.zip` 到任意目录
   ```
   例如：C:\BendAgent
   ```

2. 修改配置文件 `agent.yaml`：
   ```yaml
   backend:
     base_url: "http://你的平台地址:8090"
     ws_url: "ws://你的平台地址:8090"
   ```

3. 双击运行 `BendAgent.exe`

#### 首次运行

首次运行时会提示输入注册码：
1. 打开管理平台
2. 进入 **Agent注册码** 页面
3. 生成或复制注册码
4. 粘贴到Agent程序
5. 点击 **注册**

### 5.3 Agent状态

| 状态 | 说明 | 操作 |
|------|------|------|
| 在线 | 正常运行 | 正常使用 |
| 离线 | 未连接 | 检查网络和配置 |
| 已卸载 | 已删除 | 需重新安装 |

### 5.4 常见问题

#### Agent显示离线

1. 检查Agent程序是否运行
2. 检查网络连接
3. 检查 `agent.yaml` 配置
4. 重启Agent程序

#### 无法注册

1. 检查注册码是否正确
2. 检查注册码是否已使用
3. 联系平台管理员

## 6. 常见问题解答

### 6.1 登录问题

**Q: 忘记密码怎么办？**
> 联系平台管理员重置密码

**Q: 账号被锁定怎么办？**
> 联系平台管理员解锁

**Q: 登录显示"无权限"？**
> 检查账号角色是否有对应功能权限

### 6.2 账号问题

**Q: 流媒体账号状态显示"错误"？**
> 检查账号密码是否正确，认证码是否有效

**Q: 批量导入失败？**
> 检查Excel格式是否正确，参考模板填写

### 6.3 Agent问题

**Q: Agent显示离线？**
> 1. 重启Agent程序
> 2. 检查网络
> 3. 重新注册

**Q: 自动化启动失败？**
> 1. 确认Agent在线
> 2. 检查账号状态
> 3. 查看错误信息

### 6.4 联系支持

如遇无法解决的问题，请联系：

| 渠道 | 联系方式 |
|------|---------|
| 客服电话 | 平台管理员提供 |
| 技术支持邮箱 | 平台管理员提供 |
| 在线客服 | 平台内嵌入口 |
