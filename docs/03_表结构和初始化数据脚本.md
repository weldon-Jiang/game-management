## 1. 数据库初始化

### 1.1 创建数据库

```sql
-- 创建数据库
CREATE DATABASE IF NOT EXISTS bend_platform
    DEFAULT CHARACTER SET utf8mb4
    COLLATE utf8mb4_unicode_ci;

-- 使用数据库
USE bend_platform;
```

### 1.2 执行初始化脚本

```bash
# 方式一：命令行执行
mysql -u root -p bend_platform < db/schema.sql

# 方式二：MySQL客户端执行
source db/schema.sql

# 方式三：执行完整初始化脚本
mysql -u root -p bend_platform < docs/03_表结构和初始化数据脚本.sql
```

## 2. 表结构说明

### 2.1 商户表 (merchant)

```sql
CREATE TABLE IF NOT EXISTS `merchant` (
    `id` VARCHAR(64) NOT NULL COMMENT '主键ID',
    `phone` VARCHAR(32) DEFAULT NULL COMMENT '联系电话',
    `name` VARCHAR(128) NOT NULL COMMENT '商户名称',
    `status` VARCHAR(16) DEFAULT NULL COMMENT '状态：active-正常,expired-过期,suspended-暂停',
    `group_id` VARCHAR(64) DEFAULT NULL COMMENT '商户分组ID',
    `vip_expire_time` DATETIME DEFAULT NULL COMMENT 'VIP过期时间',
    `expire_time` DATETIME DEFAULT NULL COMMENT '过期时间',
    `is_system` TINYINT(1) DEFAULT 0 COMMENT '是否系统内置',
    `total_points` INT DEFAULT 0 COMMENT '累计充值点数',
    `vip_level` INT DEFAULT 0 COMMENT '当前VIP等级',
    `created_time` DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    `updated_time` DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    `deleted` TINYINT(1) DEFAULT 0 COMMENT '逻辑删除标记',
    PRIMARY KEY (`id`),
    KEY `idx_phone` (`phone`),
    KEY `idx_status` (`status`),
    KEY `idx_group_id` (`group_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='商户表';
```

| 字段 | 类型 | 说明 |
|------|------|------|
| id | VARCHAR(64) | 主键，使用UUID |
| phone | VARCHAR(32) | 联系电话 |
| name | VARCHAR(128) | 商户名称 |
| status | VARCHAR(16) | 状态：active/expired/suspended |
| group_id | VARCHAR(64) | 商户分组ID |
| vip_expire_time | DATETIME | VIP过期时间 |
| expire_time | DATETIME | 过期时间 |
| is_system | TINYINT(1) | 是否系统内置 |
| total_points | INT | 累计充值点数 |
| vip_level | INT | 当前VIP等级 |
| created_time | DATETIME | 创建时间 |
| updated_time | DATETIME | 更新时间 |
| deleted | TINYINT(1) | 逻辑删除标记 |

    `params` TEXT COMMENT '任务参数JSON',
    `result` TEXT COMMENT '任务结果JSON',

    `created_by` VARCHAR(64) DEFAULT NULL COMMENT '创建人',

    `started_time` DATETIME DEFAULT NULL COMMENT '开始执行时间',

    `created_time` DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',

## 4. 数据字典

| 值 | 说明 | 权限 |

| 值 | 说明 |

| expired | 已过期 |
| suspended | 已暂停 |

| 值 | 说明 |
|------|------|
